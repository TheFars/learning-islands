Массивы используются для хранения коллекций информации, но может быть полезным представить массив как коллекцию переменных одинакового типа.
Вместо объявления множества переменных и хранения в них индивидуальных значений вы можете объявить один массив для хранения всех этих значений.
При объявлении массива укажите тип его элементов, а также и количество хранимых им элементов.
Вы можете инициализировать массив, указав все его значения.

Например:
int fib[7] = {1, 1, 2, 3, 5, 8, 13};
