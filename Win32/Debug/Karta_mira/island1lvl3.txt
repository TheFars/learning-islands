Создание переменной резервирует место, или пространство в памяти для хранения значений. Компилятору необходимо, чтобы вы указали тип данных для каждой объявляемой переменной.
С++ предлагает большой ассортимент встроенных типов данных.
Целочисленный тип представляет собой целое число. Для определения переменной целочисленного типа используется ключевое слово int.
С++ требует, чтобы вы указали тип и идентификатор для каждой переменной.
Идентификатор - это имя для переменной, функции, класса, модуля или чего-либо другого, определённого пользователем.
Идентификатор начинается с буквы или нижнего подчёркивания.

Например, определим переменную myVariable, которая может хранить целочисленные значения:
int myVariable = 10;
